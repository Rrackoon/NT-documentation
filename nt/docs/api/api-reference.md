# API Reference

Интерактивная OpenAPI-документация подключена через `redocusaurus`

## Открыть спецификацию

- [Перейти к OpenAPI reference](/api/)

## Основные группы эндпоинтов

- `User` - профиль и дневная статистика
- `Meal` - приемы пищи и позиции в дневнике
- `Product` - поиск продуктов и создание пользовательских блюд
- `Weight` - история веса
- `Goal` - дневные цели по КБЖУ

## Формат авторизации

API использует `Bearer JWT`

## Источник спецификации

Исходный YAML-файл хранится в репозитории: `nt/openapi/nt-api.yaml`
