# Контекстная диаграмма

Ниже размещена редактируемая схема Draw.io. Она показывает базовый контур MVP и внешние зависимости, которые появятся или расширятся в следующих версиях

<Drawio content={contextDiagram} />

## Что показывает схема

- пользователь работает через мобильное приложение
- приложение обращается к backend API
- backend пишет транзакционные данные в PostgreSQL
- поиск продуктов опирается на индекс Elasticsearch
- временные ряды и аналитика веса хранятся в TimescaleDB
- агрегаты и frequently-read статистика могут кэшироваться в Redis
