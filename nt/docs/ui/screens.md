# Макеты экранов

## Готовые экраны

### Дневник питания

- `Route`: `myapp://home`
- Основной экран после входа
- Показывает дневной прогресс по калориям и БЖУ
- Содержит список приемов пищи за выбранную дату

Основные элементы:

- круговой индикатор калорий
- прогресс-бары по белкам, жирам и углеводам
- список приемов пищи
- нижняя навигация: дневник, профиль, аналитика

Состояния:

- Loading
- Empty
- Error

### Добавление нового приема пищи

- `Route`: `myapp://meal/add`
- Экран выбора продукта и указания количества
- После заполнения автоматически рассчитывается КБЖУ

Основные элементы:

- поиск продуктов
- фильтр категорий: любимые, недавние, часто добавляемые
- список текущих позиций в приеме пищи
- итоговый расчет КБЖУ
- кнопка `Готово`
- кнопка `+ Свой продукт`

Состояния:

- Empty
- Editing
- Error

### Профиль

- `Route`: `myapp://profile`
- Экран профиля пользователя с кратким обзором прогресса и текущих целей

Основные элементы:

- переключатель вкладок `Me/Buddies`
- карточка пользователя с именем, целью и выбранной программой питания
- блок `Обзор` с краткой сводкой текущего прогресса
- блок `Мой прогресс` с отображением текущего веса и целевого значения
- блок `Мои цели` со списком активных целей и кнопкой `Редактировать`
- нижняя навигация: дневник, профиль, аналитика

Состояния:

- Loading
- View
- Error

### Навигационная страница аналитики

- `Route`: `myapp://analytics`
- Экран выбора показателя для просмотра детальной аналитики
- Содержит набор карточек с основными и дополнительными метриками

Основные элементы:

- блок `Основные показатели`
- карточки: `Энергия из пищи`, `Рацион`, `Водный баланс`, `Расход энергии`, `Шаги`, `Вес`
- блок `Дополнительные показатели`
- карточки: `Питательные вещества`, `Минералы`, `Витамины`, `Замеры`
- аватар пользователя
- нижняя навигация: дневник, профиль, аналитика

Состояния:

- Loading
- Grid view
- Empty
- Error

### Детальная аналитика по выбранному параметру

- `Route`: `myapp://analytics/{metric}`
- Экран раскрывает статистику по конкретному выбранному показателю
- Открывается после нажатия на карточку параметра на экране аналитики

Основные элементы:

- заголовок выбранного параметра, например `Энергия из пищи`
- переключатель периода: `По дням`, `По неделям`, `По месяцам`
- сводка за выбранный период, например среднее значение и целевое значение
- график, показывающий динамику показателя
- блок `История` со списком значений по датам
- нижняя навигация: дневник, профиль, аналитика

Состояния:

- Loading
- Chart view
- Empty
- Error

## Переходы между экранами

- `myapp://home -> myapp://meal/add`
- `myapp://meal/add -> myapp://home`
- `myapp://home -> myapp://profile`
- `myapp://home -> myapp://analytics`
- `myapp://profile -> myapp://analytics`
- `myapp://analytics -> myapp://analytics/{metric}`
- `myapp://analytics/{metric} -> myapp://analytics`
