# NutriTrack Documentation

NutriTrack - это клиент-серверная система для ведения дневника питания, расчета КБЖУ, отслеживания массы тела и дальнейшего развития в сторону персонализированной аналитики рациона

## Контекст проекта

Система решает проблему фрагментарного учета питания. Вместо отдельных калькуляторов, таблиц и консультаций пользователь получает единое пространство для:

- ведения пищевого дневника
- расчета калорий и макронутриентов
- фиксации веса и просмотра динамики
- дальнейшего сопоставления рациона с медицинскими показателями
- получения персонализированных рекомендаций

## Границы MVP

В MVP включены:

- регистрация и авторизация
- дневник питания
- расчет калорий и макронутриентов
- фиксация массы тела
- базовая аналитика прогресса

Вне MVP остаются:

- анализ микронутриентов
- загрузка лабораторных показателей
- автоматические рекомендации
- онлайн-консультации
- интеграции с внешними сервисами
